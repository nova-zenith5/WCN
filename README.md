## 五、高阶优化方法论
### 5.1 返点资金循环策略
```
返点收入 = 基准值 × (实际消耗/目标消耗)^0.8
投放策略优化基金 ≥ 返点金额 × 60%
风险储备金 ≤ 返点金额 × 20%
市场情报采购预算 ≤ 返点金额 × 20%
```

### 5.2 机器学习阶段操作规范
- **0-24小时（冷启动）**
  禁用：受众修改、预算调整、出价变更
  启用：实时数据监控看板

- **24-72小时（学习期）**
  微调范围：
  ▸ 单日预算+25%/-10%

- **72小时+（稳定期）**

  ▸ 转化价值优化
